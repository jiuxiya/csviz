## 🗂️ CSV File Tips

csviz works best with clean CSV files. For smooth results:

- keep one header row at the top
- use commas between values
- avoid extra blank rows
- keep numbers in numeric columns
- use simple column names

Example CSV:

name,value
Alpha,12
Beta,25
Gamma,18

This kind of file works well for a bar chart or table.
